Технически Изисквания

  • Всички страници от проекта трябва да изглеждат еднакво на всички популярни бразлъри (последни версии): Chrome, FF, IE

  • Проекта трябва да бъде качен във собствено git репозитори и да бъде обновяван редовно

Код конвенции

  • HTML

    #ID селектори се изписват със camelCase: #exampleSelector

    .class селектори се изписват с dash-case: .example-class

    Имена на графични файлове (картинки) се изписват с underscore_case: example_file.png

    Идентацията във файловете (HTML, CSS, JS) трябва да са еднакви във всички файлове: Един таб = 4 спейса
